# 台股當沖模擬器

連結臺灣證券交易所 / 櫃買中心即時行情的當沖模擬網站，支援上市與上櫃股票。

## 啟動方式

雙擊 `啟動.bat`（會自動開啟瀏覽器），或手動執行：

```
python server.py
```

然後開啟 http://localhost:8800

> 需要 Python 3（免安裝任何套件），且電腦需能連上網際網路。

## 功能

- **股票搜尋**（右上角）：輸入代碼或名稱，支援上市（證交所）與上櫃（櫃買中心）。
- **K線圖**：日K / 5分K / 1分K 切換，附成交量與 5 / 20 / 60 / 120 均線（SMA，圖上方顯示最新值），紅漲綠跌（台股慣例）。
  - 日K：近 12 個月歷史資料（上市取自證交所、上櫃取自櫃買中心）。
  - 1分K / 5分K：歷史資料取自 Yahoo Finance（近 5 個交易日，真實成交價，每分鐘自動回補）；盤中最新分K由證交所即時行情每 5 秒合成銜接。
- **即時報價**：現價、漲跌、開高低、昨收、總量、五檔買賣報價，每 5 秒更新。
- **我的最愛**（左側欄）：點報價旁的 ☆ 加入 / 移出，清單顯示即時價與漲跌幅，點擊即可切換開啟。
- **庫存**（左側欄）：目前模擬持有的部位，即時顯示現價、損益與損益率，點擊可開啟該股。
- **損益月曆**（頂欄 📅）：以月曆檢視每日已實現損益與交易筆數，紅賺綠賠，可切換月份；資料隨帳戶保存，按重置才清空。
- 所有損益（含月曆、庫存、交易紀錄）均已內含手續費與證交稅，與帳戶現金增減一致。
- **模擬下單**：市價買進 / 賣出（可放空），單位為張。
  - 初始資金 10,000,000。帳戶、庫存、交易紀錄、月曆與我的最愛統一存在伺服器端的 `data.json`，換瀏覽器或清除瀏覽器資料都不會遺失（所有連進此網站的瀏覽器共用同一份帳戶）。
  - 手續費 0.1425%（低消 20 元）、賣出收當沖證交稅 0.15%。
  - 顯示部位、均價、未實現 / 已實現損益與交易紀錄，可一鍵重置。

## 資料來源與限制

| 資料 | 來源 |
|---|---|
| 即時報價 / 五檔 | `mis.twse.com.tw/stock/api/getStockInfo.jsp` |
| 分K歷史 | Yahoo Finance chart API（證交所無公開歷史分K） |
| 股票代碼搜尋 | 證交所 codeQuery + 櫃買中心 OpenAPI 上櫃清單 |
| 日K（上市） | 證交所 `STOCK_DAY` |
| 日K（上櫃） | 櫃買中心 `afterTrading/tradingStock` |

- 證交所匿名即時端點會遮蔽個股「最新成交價」欄位，本站在成交量增加（確實有成交）時以買一 / 賣一中價推估成交價來合成分K；開高低、量、五檔皆為交易所原始資料。
- 即時行情約有數秒延遲，本模擬器僅供練習參考，非投資建議。

## 部署成公開網站

GitHub Pages 只能放靜態頁面、無法執行 Python 伺服器（本站需要伺服器代理證交所 API 以繞過 CORS），因此建議用免費的 [Render](https://render.com) 搭配 GitHub 自動部署：

1. 把這個資料夾推上 GitHub（`data.json` 已在 `.gitignore`，個人紀錄不會上傳）。
2. 到 render.com 用 GitHub 帳號登入 → **New → Web Service** → 選這個 repo。
3. Render 會讀取 `render.yaml` 自動設定（免費方案、`python server.py` 啟動），按下 Deploy 即可。
4. 完成後會得到 `https://<名稱>.onrender.com` 的公開網址。

雲端上會自動進入**公開模式**：每位訪客的帳戶與最愛存在自己的瀏覽器（localStorage），彼此不互通；本機執行則維持存 `data.json`。

> 注意：免費方案閒置 15 分鐘會休眠，下次開啟需等約 30 秒喚醒。即時行情僅在台股盤中（09:00–13:30）有變化。

## 檔案結構

| 檔案 | 說明 |
|---|---|
| `server.py` | 本機伺服器：靜態網頁 + 證交所/櫃買 API 代理（解決 CORS） |
| `index.html` / `style.css` / `app.js` | 前端頁面（K線使用 TradingView lightweight-charts） |
| `啟動.bat` | 一鍵啟動 |
| `data.json` | 帳戶/最愛紀錄（自動產生，備份此檔即備份紀錄） |
